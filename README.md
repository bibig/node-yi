# Yi
+ this module collect common useful functions
+ btw, yi is a chinese word mean easy.

### install
 npm install yi

### members

+ isEmpty
+ clone
+ merge
+ filter

### usage

``` javascript
  var yi = require('yi');

  yi.isEmpty(undefined); // true
  yi.isEmpty(null); // true
  yi.isEmpty(''); // true
  yi.isEmpty([]); // true
  yi.isEmpty({}); // true
```
